# DayiExt
A chrome extension for dayi.im

* [Usage](#usage)
* [Function](#function)
* [Download](#download)
* [Reference](#reference)

---

##Usage:

打开 `chrome://extensions/`

选择 `开发者模式`

选择`加载正在开发的扩展程序`

打开下载的文件夹

---

##Function:

按图标开始刷新(我想把这改成按键刷新 但是在content里获取了按键不知到怎么传递信息 求腿带飞)

`快速抢题`移到button上就能抢题

`快速按键抢题`1对应第一题 2对应第二题... 只映射了1~9

预约的`快速接受`和`快速拒绝`

修复网站的一些错误链接

限制了图片的高度

不同价钱的题目不同的标题颜色
